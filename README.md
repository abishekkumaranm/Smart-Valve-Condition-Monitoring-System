# Smart Valve Condition Monitoring System

## 🔍 Overview
This project presents an IIoT-based smart monitoring system for industrial valves to enable real-time condition monitoring and predictive maintenance.

## 🚨 Problem
Industrial valves often fail without early warning, leading to:
- Unplanned downtime
- Safety risks
- High maintenance costs

## 💡 Solution
Developed a smart monitoring system using sensors and IIoT architecture to continuously track valve health.

## ⚙️ System Architecture
- Sensor Layer (Pressure, Temperature, Vibration, Position)
- Edge Gateway (Data acquisition & processing)
- Monitoring Dashboard (Visualization & alerts)

## 📊 Features
- Real-time sensor monitoring  
- Predictive maintenance alerts  
- Valve health score analysis  
- Historical data tracking  

## 🛠️ Technologies Used
- Sensors: PT100, Pressure sensor, MEMS accelerometer  
- Communication: 4–20 mA, Modbus  
- Platform: Web Dashboard
